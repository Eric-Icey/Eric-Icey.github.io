操作系统

一 操作系统概述

(一). 初步认识:

  1. 没有操作系统依旧可以正常使用程序
    • 只能使用刻录在计算机硬件上的bios程序
    • 操作系统与普通软件有重大差别
    • 例如在蓝屏界面设置计算机基本信息时
    • 此类程序的编写与安装都需要特殊的方式
  2. 普通用户无法运行正常软件
  3. 操作系统提供的功能:
    1. 可视化操作界面；
    2. 控制普通程序运行；
    3. 控制多个程序并发运行，管理系统资源；
    4. 进行系统设置/配置，监控系统状态；
    5. 提供工具软件集合；
    6. 信息的存取及其保护机制；
  4. 操作系统对应用程序运行的支持：
    1. 存放.exe文件；
      • 提供对应的接口，将文件放在特定的硬盘目录下；
    2. 启动.exe文件；
    3. 为程序分配内存；
      • 待解答;
    4. 为程序提供输入/输出的驱动/硬件接口；
    5. 如何处理程序死循环对CPU的影响？
      • 由于分时系统的存在，该循环程序只会在获得CPU资源时有效，其余时间内其它程序仍能正常运行
    6. 程序如何退出系统？
      • 待解答;
  5. 课后问题
    1. 没有安装操作系统的计算机启动过程和结果？
      • 启动：启动后直接显示为DOS或者时BIOS界面
      • 过程：使用简单的键盘进行计算机基本设置的配置，比如修改参数一类
      • 结果：仅能修改计算机本身的属性，但无法进行其他更为高级的设置
    2. 常见的虚拟机软件(VMware VirtualPC)能不能理解为操作系统？
      • 不能。
      • 虚拟机软件只是为操作系统提供了一个可以安装的地方，其本质应该也是属于普通计算机的范畴，其所有的硬件设置都是虚拟出来的，拥有其对应的参数。如需正常使用仍需额外安装操作系统。
